# Habitat EUNIS A5.531 : Herbiers de Cymodocea

#### 🔍 Informations descriptives
Formations de Cymodocea nodosa sur les côtes de l’Atlantique, dans le sud de la péninsule ibérique, le nord‐ouest de l’Afrique et les îles macaronésiennes.
