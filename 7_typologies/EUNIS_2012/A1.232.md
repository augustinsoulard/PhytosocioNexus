# Habitat EUNIS A1.232 : Concrétion de Neogoniolithon brassica-florida

#### 🔍 Informations descriptives
L’espèce caractéristique de cette association est l’algue rouge Neogoniolithon brassica‐florida. L’association occupe, lorsqu’elle est présente, les bordures du médiolittoral inférieur, en mode battu.
